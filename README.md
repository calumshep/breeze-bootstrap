## breeze-bootstrap

This template repository provides a starting point for a Laravel application with a Bootstrap frontend.

## Requirements

- PHP 8.2 or greater

## Software Installed

- Laravel 10.10 or greater
- PHPUnit 10.1 or greater
- Bootstrap 5.3.1 or greater
